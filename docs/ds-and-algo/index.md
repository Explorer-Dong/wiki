!!! tip
    数据结构与算法是程序的灵魂。

本专栏以「数据结构与算法」笔记为主。参考杜老师的 [冠军之路学习树](https://zhuanlan.zhihu.com/p/454647571) 组织内容。欢迎在评论区留言补充🤗。

本专栏题目主要源自以下几个平台：

- [Codeforces](https://codeforces.com/)：毛子开发的刷题平台，适合高难度训练。
- [洛谷](https://www.luogu.com.cn/)：国内 OI 选手参与较多，也适合入门。
- [LeetCode](https://leetcode.cn/)：求职使用较多，题型相对固定。
- [灵茶の试炼](https://docs.qq.com/sheet/DWGFoRGVZRmxNaXFz?tab=BB08J2)：著名刷题大佬收集的题单，适合补题。

本专栏内容安排如下：

- 封装套路代码，提升编码速度：[C++ 代码模板](./templates.md)、[Python 代码模板](./templates-py.md)。
- 剖析算法原理，速通经典例题：[专题解析](./topic/index.md)。
- 解析真实赛题，感受实战魅力：[蓝桥杯备赛指南](./lan-qiao-cup/index.md)。

一些好用的工具：

- [Graph Editor](https://csacademy.com/app/graph_editor/)：简易有/无向图绘制工具，可以根据数据很方便地绘制 Graph。
- [CFTracker](https://cftracker.netlify.app/contests)：Codeforces 追踪器，展示每场比赛的每道题目分数，便于按难度补题。
