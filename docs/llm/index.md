本专栏围绕全模态大模型生态展开介绍，欢迎评论与补充🤗。主要包括以下三个部分：

1. [**基础设施**](./infra/index.md)：包括硬件生态、数据系统、训练框架、推理框架等；
2. [**模型原理**](./model/index.md)：包括模型架构、模型预训练、模型后训练等；
3. [**下游应用**](./downstream/index.md)：包括代码智能体、智能体编排工具、检索生成增强器、提示词工程等。

大模型的一些资源：

- 顶级大模型：
    - 国外御三家：[ChatGPT](https://chatgpt.com/)、[Gemini](https://gemini.google.com/app)、[Claude](https://claude.ai/new)。
    - 国内第一梯队：[DeepSeek](https://chat.deepseek.com/)、Qwen、豆包、Minimax、智谱。

- API 供应商：
    - 各模型官网的开放平台：稳定可靠，但国外大模型支付有门槛。
    - [OpenRouter](https://openrouter.ai/)：头部中转站，费用较高，较高的并发有累计充值门槛。
    - [硅基流动](https://siliconflow.cn/zh-cn/models)：只支持部分模型。
    - [阿里云百炼](https://bailian.console.aliyun.com/#/home)：只支持阿里系模型。
    - [七牛云](https://portal.qiniu.com/home)、[七牛云支持的 Model](https://sufy.com/zh-CN/services/ai-inference/models)：模型列表比官网更全，懂得都懂。
    - [PlanetZero API](https://api.planetzero.live/)：并发性不错，模型也比较全，价格也比较低，但稳定性不能保证。

- 主流应用：
    - 代码智能体：[Cursor](https://cursor.com/)、Claude Code、Codex。
    - 智能体编排工具：[Dify](https://dify.ai/zh)、[扣子](https://code.coze.cn/home)。
    - 检索生成增强器：[RAGFlow](https://github.com/infiniflow/ragflow)、[RAGFlow 知乎官号，博客质量较高](https://www.zhihu.com/org/infiniflow)。
- 大模型榜单：[Artificial Analysis](https://artificialanalysis.ai/)
