## 内容简介

本项目托管笔者的技术博客，围绕「计算机」与「人工智能」两大主题，展开以下四个板块：

- **基础知识**：人工智能、计算机、数学等基础知识；
- **数据结构与算法**：数据结构、动态规划、图论等经典算法；
- **软件开发**：前端、后端、运维等开发方法；
- **大模型**：基础设施、模型架构、下游应用等研究内容。
