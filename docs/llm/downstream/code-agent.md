## Claude Code

在 WSL/Linux 环境中操作。

安装 [Open Code](https://opencode.ai/docs/windows-wsl)：

```bash
curl -fsSL https://opencode.ai/install | bash
```

安装 [Claude Code](https://code.claude.com/docs/en/overview)：

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

安装 [iFlow](https://platform.iflow.cn/cli/quickstart)：

```bash
bash -c "$(curl -fsSL https://gitee.com/iflow-ai/iflow-cli/raw/main/install.sh)"
```
